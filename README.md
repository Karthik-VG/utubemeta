# utubemeta

- Utubemeta is a app which helps to scrap the metadata for youtube videos.It will scrap title,description,tag,channel name and views.These metadata will helpful in building tag generator for youtube videos..etc


# STEPS

## step 01 - Clone the new repository

## step 02 - Create a conda environment after opening the repository in VSCODE or any IDE
```
conda create --prefix ./env python=3.7 -y
```
## activate environment
```
conda activate ./env
```
### or
```
source activate ./env
```

## STEP 03- install the requirements
```
pip install -r requirements.txt
```
## step 04- run main.py

```
python main.py 

```
